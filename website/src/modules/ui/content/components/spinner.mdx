# Spinner

Used for the loading state of pages. When part of the page is waiting for asynchronous data or is in the rendering process, appropriate loading amimation will effectively alleviate user anxiety.

## Import

```javascript
import { Spinner } from '@resolid/ui';
```

## Sizes

Alert has 5 sizes you can use. Pass the `size` prop and use either `xs`, `sm`, `md`, `lg` or `xl`.

<Demo demo={'spinner/SpinnerSizes.tsx'} />

## Colors

Change the color of the alerts by passing the `color` prop. This affects the color scheme and icon used. Alert supports `primary`, `success`, `warning`, `danger`, and `neutral` colors.
